# FUS 14기 Git 수업
## 성수캠퍼스
11월 16일부터 20일까지 패스트캠퍼스 성수교육장에서 Git 관련 수업 내용을 정리해 봤습니다.
로컬 저장소와 리모트 저장소를 관하는 방법도 학습했습니다.

## CLI
- pwd 현재 작업중인 폴더 확인
- whoami 루트이름 확인
- clear 창 지우기
- mkdir 폴더 생성하기
- cd 폴더 경로
- ls 폴더 내 목록 확인
- rmdir 폴더 지우기
- rm 파일/내용이있는 폴더 삭제
- mv 이름 바꾸기/ 폴더 이동하기

## Git
- 초기환경 설정
   - git config --global user.name
   - git config --global user.email
 - git init
 - git status
 - git add
 - git commit -m (커밋 메시지 작성)
